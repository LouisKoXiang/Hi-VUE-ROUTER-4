# HI-VUE3

1. option api -> composition api
2. vue-router3 -> vue-router4
3. vee-validate 3.0 -> vee-validate 4.0
4. vuex 3.0 -> 4.0
5. Vue2 Instal -> Vue3 mount
6. ...
